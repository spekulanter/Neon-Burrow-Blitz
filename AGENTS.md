# NEON BURROW BLITZ

## Workflow

- Work asset-first: verify PNG dimensions, import settings, slicing, cleanup, processed frames, animations, and prefabs before gameplay scene work.
- Do not use internet or paid external packages.
- Prefer a compiling, playable Unity prototype over visual perfection.
- After each major phase, refresh Unity and check the console for compile errors.
- Preserve unrelated user or Unity-generated changes already present in the worktree.

## Unity

- Project: Unity 6000.3, URP 2D.
- Main preparation entry point: `Tools/Neon Burrow/Prepare Project`.
- Raw art belongs in `Assets/Art/Raw/`.
- Generated art belongs in `Assets/Art/Processed/`.
